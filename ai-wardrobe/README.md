# Jarvis Wardrobe

可直接開啟的 AI 智慧衣櫃前端原型。

正式網址：<https://jarvis-wardrobe.pages.dev/>

## 功能

- 上傳多張衣物照片並標記分類
- 依分類瀏覽、刪除衣物
- 依色彩、場合正式度、溫度材質、風格與明暗平衡評分
- 從全部候選組合中挑選三套高分穿搭，並顯示推薦理由
- 收藏穿搭
- 使用 `localStorage` 保留衣櫃與收藏，照片不會上傳伺服器
- 響應式桌面／手機版面
- 手機底部導覽與中央拍攝入口
- 支援安裝到 iPhone／Android 主畫面的 PWA
- 部署後可離線開啟與使用既有衣櫃

## 本機預覽

在此資料夾啟動任意靜態伺服器，例如：

```powershell
python -m http.server 4173
```

接著開啟 `http://localhost:4173`。

手機使用需要先部署到 HTTPS 網址。開啟網址後，可使用瀏覽器的「加入主畫面」功能，之後會以接近原生 App 的全螢幕模式啟動。

## 後續正式版

目前會從照片預估主色，材質、正式度與風格由使用者確認。正式產品可再接上視覺模型自動辨識品類、材質與版型，並加入身形、行程、即時天氣及穿著回饋作為推薦特徵。
